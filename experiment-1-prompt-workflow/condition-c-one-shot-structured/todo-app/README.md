# Todo App

A simple and clean Todo application built with React.

## Features

- ✅ Add new tasks
- ✅ Delete tasks
- ✅ Mark tasks as complete/incomplete
- ✅ Persist tasks in browser localStorage
- ✅ Display list of tasks

## Installation

```bash
npm install
```

## Development

```bash
npm run dev
```

Open your browser and navigate to `http://localhost:5173`

## Build

```bash
npm run build
```

## Technology Stack

- React 18
- Vite
- CSS3
